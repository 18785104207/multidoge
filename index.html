    Welcome to MultiDoge

    MultiDoge is a thin client Dogecoin wallet. It's a port of the MultiBit client for Bitcoin.

    About

    The app is based on DogecoinJ, which in turn is a port of BitCoinJ (@bitcoinj). You can find DogecoinJ here: https://github.com/langerhans/dogecoinj-new I ported all the needed changes for Dogecoin compatibility over. This program uses a special branch of it, which can be found here.

    Building

    Basically you clone both this repo and DogecoinJ and compile it with Maven. More info is included in the README file.
